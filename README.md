# Travel Buddy

## Overview
Travel Buddy is an Android application that helps users explore tourist destinations, view travel information, and plan trips efficiently.

## Features
- Explore Places
- Destination Categories
- Travel Tips
- User-Friendly Interface
- Navigation Drawer
- Attractive UI

## Technologies Used
- Java
- XML
- Android Studio
